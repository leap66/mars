
*** ClearTextField ***

* 带左侧label标题右侧清空按钮的输入框控件

* 内嵌ClearEditText（不带左侧Label标题的输入框控件，只支持右侧清空）可单独使用

# attrs属性说明
# label 左侧标题栏
-- app:label_text   标题栏文字
-- app:label_text_size  标题栏文字大小
-- app:label_text_color 标题栏文字颜色
-- app:label_layout_width   标题栏宽度
-- app:label_padding    标题栏内部间距
-- app:label_padding_left   标题栏内部左间距
-- app:label_padding_top    标题栏内部上间距
-- app:label_padding_right  标题栏内部右间距
-- app:label_padding_bottom 标题栏内部下间距
-- app:label_gravity 文字位置

# edit 输入框
-- app:showClear 是否显示清除按钮（默认显示）
-- app:edit_text 输入框文字
-- app:edit_text_size   输入框文字大小
-- app:edit_text_color  输入框文字颜色
-- app:edit_maxLength  输入框文字长度
-- app:edit_hint    输入框提示文字
-- app:edit_hint_color  输入框提示文字颜色
-- app:edit_left_drawable   输入框左侧图片
-- app:edit_padding  输入框内部间距
-- app:edit_padding_left    输入框内部左间距
-- app:edit_padding_top     输入框内部上间距
-- app:edit_padding_right   输入框内部右间距
-- app:edit_padding_bottom  输入框内部下间距
-- app:edit_errMsg  错误提示信息
-- app:edit_gravity 文字位置
-- app:edit_inputType 输入类型
